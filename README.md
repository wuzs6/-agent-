# 多智能体协同工作管理系统
基于多 Agent 协同的日常工作自动化管理工具，支持任务分配、进度跟踪、智能提醒、自动总结。

## 功能介绍
-  **任务分配Agent**：自动分配任务负责人
-  **进度跟踪Agent**：实时监控任务状态，自动标记逾期
-  **智能提醒Agent**：6小时内即将截止/已逾期任务自动提醒
-  **总结汇报Agent**：一键生成每日工作汇报
-  **中央协调Agent**：统一调度所有Agent，实现全自动工作流

## 快速开始
### 环境要求
Python 3.6+

```bash
python workflow_agent.py
