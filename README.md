# High Throughput Multi-Agent Work Management System
高吞吐多智能体协同工作管理系统（大模型版）

## 项目介绍
本项目是企业级多智能体工作管理平台，专为**大模型高并发、高 Token 消耗场景**设计，支持任务自动化解析、分配、跟踪、风险预警、提醒与日报生成。

系统由 7 大智能体协同工作，所有决策与文本生成均通过大模型完成，可稳定实现 **日均 3,000,000 Token 消耗**。

## 核心智能体
- 任务解析智能体
- 任务分配智能体
- 进度跟踪智能体
- 风险预警智能体
- 自动提醒智能体
- 工作日报生成智能体
- 中央协调中枢

## Token 消耗设计（日耗 300 万）
- 单次全流程消耗：20,000 Token
- 每日执行次数：150 次
- 日消耗总量：3,000,000 Token
